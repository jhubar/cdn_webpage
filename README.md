# CDN Automobiles — site vitrine

React + Vite + Tailwind CSS.

## Prérequis Node

Si vous utilisez nvm ou une config dans `~/.zshrc`, chargez-la avant `npm` :

```bash
source ~/.zshrc
```

## Commandes

```bash
npm install
npm run dev
npm run build
npm run preview
```

Les sections utilisent des images placeholder (Unsplash) à remplacer par vos photos CDN Automobiles.
